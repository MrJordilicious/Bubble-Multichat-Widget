📋 Setup Instructions
  1. In Streamer.bot go to Servers/Clients → WebSocket Server and enable it (default port 7474)
  2. Add a Browser Source in OBS and paste the generated URL below
  3. Set the browser source size to your canvas (e.g. 1920×1080)
  4. Check "Allow transparency" in the browser source settings
  5. Leave Custom CSS empty
  6. For Shared Chat: make sure Streamer.bot has the other channel connected
